# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, email us at security@levolink.com with:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

We will respond within 48 hours and work with you to resolve the issue.

## Security Measures

- All API traffic uses HTTPS/TLS encryption
- API keys are stored with industry-standard hashing
- No API keys are logged or persisted in plain text
- Regular security audits of infrastructure

## Responsible Disclosure

We follow responsible disclosure practices. Security researchers who report valid vulnerabilities will be acknowledged (if desired) and may receive rewards.
